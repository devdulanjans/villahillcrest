Next.js migration scaffold for Villa Hillcrest

Quick start:

1. Move your image assets into `next-app/public/images` (create the folder and copy files from the root `images/`).
2. From `next-app` run:

```bash
npm install
npm run dev
```

Notes:
- The scaffold uses the Pages router for simplicity. It places global CSS in `styles/globals.css`.
- Keep the original content in the root for reference. Update components in `pages` as needed.
- After you verify, you can move this app to the repository root or a separate repo.
